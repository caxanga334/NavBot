# Day of Defeat: Source Mod Settings

These settings are only used in Day of Defeat: Source.

| Key | Description | Range | Default |
|:---:|:---:|:---:|:---:|
| dod_bomb_plant_respond_dist | Maximum distance between the bot and a bomb for then to care about defusing it. | 1000 - 10000 | 3000 |
| dod_bomb_max_defusers | Maximum number of bots allowed to respond to a planted bomb. | 2 - 16 | 3 |
