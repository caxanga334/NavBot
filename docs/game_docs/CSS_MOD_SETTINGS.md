# Counter-Strike: Source Mod Settings

These settings are only used in Counter-Strike: Source.

| Key | Description | Range | Default |
|:---:|:---:|:---:|:---:|
| eco_limit | If the bot has less money than this, do a full eco. | 1000 - 4000 | 2000 |
| team_eco_limit | If the average team money is less than this, do a full eco. | 500 - 4500 | 1500 |
| timeleft_to_hurry | If the time left in the round is less than this, bots will 'hurry'. | 0.0 - 60.0 | 40.0 |
